# Macau Tour Map

Mobile-friendly one-day Macau route map for a trip from Hengqin Port to Taipa Ferry Terminal.

Open the published GitHub Pages site on a phone:

https://sergioyin.github.io/macau-tour-map/

## Files

- `index.html`: published map page
- `macau-navigation-routes.html`: same navigation map with route details
- `macau-navigation-routes-mobile-preview.png`: mobile screenshot preview
- `tools/build_macau_navigation_map.mjs`: source script that rebuilds the HTML from route data
